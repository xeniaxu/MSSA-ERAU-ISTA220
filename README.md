## The program leverages college accredited courses and the goal of preparing transitioning service members for technical opportunities in the IT industry utilizing the following skills:
* ‚òë Studying the software development lifecycle and how to use it to successfully develop various console-based and graphical applications using C# and Microsoft Visual Studio 2019.
* ‚òë Learning how to use T-SQL to create, retrieve, update, and delete data inside of databases using Microsoft SQL Server Management Studio 2019.
* ‚òë Will complete 360 classroom hours learning and mastering the fundamentals of C#, T-SQL, ASP. Net MVC, and Azure.
* ‚òë Will use ASP.Net MVC as well as HTML and CSS to create multiple web-based applications.
